#Zthings_IOSAPP

##Intro
这是一个用GAE with Python 作为后台， 用iPhone App作为前端的架构。Zthings_IOSAPP.git 只是iPhone APP部分

GAE with Python的后端 是[Z IN THE DREAM](http://zinthedream.appspot.com), 源码可以在是[zths.git](http://github.com/mengfeng/zths.git)
